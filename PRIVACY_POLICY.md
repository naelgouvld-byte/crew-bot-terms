## 3. Données Collectées

### 3.1 Données Automatiquement Collectées
Le Bot collecte automatiquement les données suivantes :

| Données | Description | Raison |
|---------|-------------|--------|
| **User ID** | Identifiant Discord unique de l'utilisateur | Identification et stockage des préférences |
| **Server ID** | Identifiant du serveur Discord | Gestion des configurations par serveur |
| **Timestamp** | Date et heure d'utilisation | Historique des activités |
| **Commandes utilisées** | Liste des commandes exécutées | Amélioration du Bot et statistiques |
| **Paramètres de configuration** | Réglages définis par les administrateurs | Sauvegarde des préférences |

### 3.2 Données Optionnellement Collectées
- Les données saisies volontairement via les commandes du Bot
- Les réactions aux messages du Bot
- Les interactions utilisateur spécifiques

### 3.3 Données NON Collectées
Le Bot ne collecte **jamais** :
- Les messages privés des utilisateurs
- Les contenus de messages publics (sauf si mentionné explicitement)
- Les adresses IP
- Les informations de localisation
- Les informations biométriques

## 4. Utilisation des Données

### 4.1 Finalités du Traitement
Vos données sont utilisées exclusivement pour :

1. **Fonctionnement du Bot** : Exécuter les commandes et services demandés
2. **Personnalisation** : Sauvegarder vos préférences et configurations
3. **Amélioration du service** : Analyser l'utilisation pour améliorer les fonctionnalités
4. **Maintenance technique** : Détecter et corriger les bugs
5. **Sécurité** : Prévenir les abus et les activités malveillantes
6. **Conformité légale** : Répondre aux obligations légales

### 4.2 Base Légale
Le traitement de vos données repose sur :
- Votre consentement implicite lors de l'utilisation du Bot
- Notre intérêt légitime à fournir et améliorer le service
- Le respect de nos obligations légales

Si des données sont transférées internationalement, elles bénéficient de protections équivalentes conformément au RGPD.

## 6. Sécurité des Données

### 6.1 Mesures de Sécurité
Nous implémentons les mesures suivantes :

- ✅ **Chiffrement** : Les données sensibles sont chiffrées en transit et au repos
- ✅ **Authentification** : Accès restreint aux données avec authentification forte
- ✅ **Audit** : Journalisation de tous les accès aux données
- ✅ **Sauvegarde** : Sauvegardes régulières pour éviter la perte de données
- ✅ **Pare-feu** : Protection contre les accès non autorisés

### 6.2 Limitations
Bien que nous prenions des mesures raisonnables, **aucun système n'est 100% sécurisé**. Nous ne pouvons garantir une sécurité absolue.
